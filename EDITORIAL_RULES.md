# Editorial Rules

These rules apply to quiz content, especially rule-based content and annual rule-change items.

## Source Priority

1. Official rulebook
2. Official rule-change document
3. Official governing-body guidance
4. Secondary sources such as articles, Reddit threads, videos, or commentary

Secondary sources may help with framing or prioritization, but they do not override the official source.

## Core Principle

Content may be rewritten for readability, but not at the cost of precision.

Do not remove:
- a condition
- a time boundary
- a positional boundary
- a scope qualifier
- an enforcement qualifier
- a terminology distinction that matters in the official source

If the official rule includes a limiting detail, keep it.

Examples of details that must not be dropped:
- `up until the next serve`
- `by voice, hand signal, or both`
- `during a live rally`
- `at least one buttock in contact`
- `even if no one is hit`

## Rewrite Standard

Allowed:
- shorter sentence structure
- cleaner word order
- less awkward phrasing
- removal of redundancy that does not change meaning
- more everyday or on-court wording when precision is fully preserved

Not allowed:
- replacing a precise rule with a looser summary
- removing a limiting clause to make the sentence shorter
- using commentary language in place of rule language
- borrowing a simplification from a media article if the official source is more precise
- replacing precise wording with more casual wording if a condition, limit, or exception is lost

## Rule-Change Items

For annual change items:
- the official rulebook and change document always win
- media coverage can help identify what is notable
- media wording should not replace official wording when it reduces precision

When in doubt:
- keep the more precise version
- move extra context into the explanation, not the question

## Media Inspiration

Media articles can be useful for:
- more natural player-facing wording
- clearer on-court phrasing
- making the practical outcome of a rule easier to understand

Media articles must not be used for:
- replacing the official rule source
- removing conditions, limits, or exceptions
- turning a precise rule into a looser summary
- importing hype, opinion, or enforcement speculation into quiz copy

Safe use of media inspiration:
- prefer stronger verbs
- prefer on-court wording over bureaucratic wording
- prefer clear consequence language such as `wins the rally` or `replay` when that consequence is already supported by the official source
- keep the same factual meaning as the official source

## Review Mode

When reviewing an existing question set, use only two categories:

- `OK`
- `Reword`

For each `Reword` item:
- explain briefly why it needs rewording
- keep the same factual meaning
- do not add new conditions, examples, or claims unless they are supported by the official source already being used

Do not invent:
- new examples
- new edge cases
- stronger wording than the source supports
- simpler wording if that simplification drops a condition, exception, or limit

Preferred review order:
- by category first
- then by full-set consistency pass if needed

Reason:
- category review makes it easier to compare similar items
- it reduces accidental drift in wording standards
- it is safer than reviewing a large mixed set in one pass

## Working Preference

Fast reading is good.

But for rule content, precision wins over punch.

The target is:
- quick to read
- easy to parse
- still faithful to the official source
- natural enough to sound like player language on court

## False-Item Standard

For items whose correct answer is `false`:
- prefer a false affirmative statement
- avoid building the item around `not`, `never`, `may not`, `must not`, or similar negative phrasing when a cleaner false affirmative is available
- avoid double-negation logic
- avoid wording traps that test sentence parsing more than rule knowledge

Preferred:
- `Players may ask spectators for help on a call.`
- `Play may continue after a fault.`
- `Letting the ball roll off the paddle face before the serve is a fault.`

Avoid when a cleaner option exists:
- `Players must not ask spectators for help on a call.`
- `A player may not let the ball roll off the paddle face before serving.`
- `A receiving team can never score a point in standard scoring.`

Exception:
- if the official rule itself is best understood through a negative boundary, a negative term may remain
- but the default target is still `false = false affirmative statement, easy to parse`
