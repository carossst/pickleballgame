Pickleball-only rollout notes

This file tracks changes implemented on Pickleball that have not been propagated to Coffee or Word Traps yet.

Current Pickleball-only items

1. Paywall redesign
- New paywall headline:
  - `You've got the feel for it. Now finish the set.`
- Hook/projection/price/CTA reordered for a shorter mobile funnel.
- `Pay once. No subscription.` promoted near the top.
- Compact single feature section added:
  - all 200 questions
  - explanations
  - Mistakes Mode + Rapid Fire
  - works offline
- Secondary `Not now` changed from button to text link.
- Testimonials stars styled in yellow.
- Paywall copy made easier to select/copy.

2. Top blue bands
- Darker top band on landing.
- Darker top band on END.
- Darker top band on paywall.
- Lighter top band on PLAYING for continuity without overpowering gameplay.
- Token-driven and intended to work in dark mode.

3. Pickleball-only loading humor
- Slow-load lines:
  - `Arguing politely about the kitchen...`
  - `Reviewing highly suspicious line calls...`
  - `Preparing an unnecessary Erne...`
- Used for boot slow loading.
- Used for content-loading messages before questions are ready.

4. Support/contact modal
- Contact modal now offers 3 direct email paths:
  - `Bug report`
  - `Question`
  - `Idea`
- Each path opens a prefilled `mailto:` draft with app-specific subject/body.
- Existing `Copy email` and generic `Open email app` remain.

5. Paywall reassurance wording
- Checkout reassurance currently:
  - `Payment handled securely by Stripe. Usually about 30 seconds.`
- Device reassurance remains:
  - `One-time unlock for this device. No account needed.`

6. Score / END readability pass
- Larger circular centered score.
- Larger END text intended to be easier to read for older players.
- END visual family pushed closer to landing.

Before propagating to other apps
- Re-check visual balance on mobile and dark mode.
- Decide whether the Pickleball paywall structure should become the cross-app standard.
- Decide whether the top blue band should be identical or theme-adjusted in Coffee and Word Traps.
